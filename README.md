# Bikeshare Project

### Date created
05-05-2024

### Description
This project uses data provided by *Motivate*, a bike share and micromobility industry.
In the project, data from 3 major US cities is analysed and displayed as a interactive experience for the user.

### Files used
- chicago.csv
- new_yor.csv
- washington.csv
